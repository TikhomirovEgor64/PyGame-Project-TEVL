Тема: Мини-игры для двоих. Змейка, танчики, понг, мемори.
Проект Тихомирова Егора и Васильева Леонида.

На главном окне будут расположены кнопки 'Игры', 'Правила'.

При нажатии кнопки 'Игры' будут предоставлены игры для выбора (змейка, танчики, понг, мемори).
После выбора игры будет запускаться соответствующая игра.

При нажатии кнопки 'Правила' будет открываться окно с таблицей, содержащей список всех игр. В последнем столбце каждой строчки таблицы будут расположены кнопки, после нажатия на данные кнопки, будет открываться окно с правилами для соответствующей игры. Правила для всех игр будут хранится в отдельных файлах.

В програме будет использоваться библиотека sqlite3. Это понадобится для таблицы с правилами.
